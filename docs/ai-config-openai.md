# AI 模型配置指南（OpenAI 及兼容服务）

> **文档职责**：说明如何在 LocalMiniDrama 中配置 OpenAI 及兼容接口的文本、图像、视频服务，并提供推荐模型清单
> **适用场景**：首次配置 AI 服务、更换模型、接入国内中转服务时参阅
> **目标读者**：普通用户，了解 API Key 的基本概念即可
> **维护规范**：AI 服务商更新模型列表时同步更新"推荐模型"节；新增支持的服务类型时同步更新"服务类型说明"节

---

## 配置入口

启动服务后，访问 [http://localhost:3013](http://localhost:3013)，点击右上角 **「AI 配置」** 按钮，进入配置页面。

---

## 服务类型说明

本平台将 AI 能力拆分为五种独立服务类型，每种类型需单独配置：

| 服务类型 | 用途 | 必须配置 |
|----------|------|:--------:|
| **text**（文本） | 故事生成、剧本扩写、角色/场景/道具自动提取、分镜脚本生成 | ✅ 必须 |
| **image**（图像） | 角色图、场景图、道具图的生成 | 推荐 |
| **storyboard_image**（分镜图） | 分镜帧图生成（支持参考图输入） | 推荐 |
| **video**（视频） | 每个分镜的短视频生成 | 推荐 |
| **tts**（语音） | 分镜配音、旁白合成 | 可选 |

> 最低可用配置：只配置 **text** 即可完成剧本创作；配置 **image + video** 后可完成完整视频制作。

---

## 如何添加一个配置

1. 点击「AI 配置」→「+ 添加配置」
2. 选择**服务类型**（如 text）
3. 选择**厂商**（如 OpenAI），系统自动填入 Base URL
4. 填入 **API Key**
5. 选择**默认模型**
6. 点击「测试连接」验证配置是否有效
7. 勾选「设为默认」→「保存」

| 字段 | 填写内容 |
|------|----------|
| 服务类型 | `text` |
| 厂商 | `OpenAI` |

| 默认模型 | 见下方推荐模型 |

| 字段 | 填写内容 |
|------|----------|
| 服务类型 | `image` 或 `storyboard_image` |
| 厂商 | `OpenAI` |

| 接口规范 | `OpenAI 兼容` |
| 默认模型 | `dall-e-3` |

## 国内中转接口配置（无法直连 OpenAI 时）

如果无法直连 `api.openai.com`，可使用国内的 OpenAI 兼容中转服务：

| 字段 | 填写内容 |
|------|----------|
| 服务类型 | `text` |
| 厂商 | 手动输入自定义名称，如 `OpenAI中转` |
| Base URL | 你的中转服务地址，如 `https://your-proxy.com` |
| API Key | 中转服务提供的 Key |
| 接口规范 | `OpenAI 兼容` |
| 默认模型 | `gpt-4o` 或中转支持的模型名 |

> 注意：使用中转服务意味着你的 API Key 会发往中转站服务器，请选择可信赖的服务商。

## 并发设置建议

进入「AI 配置」→「生成设置」调整并发数：

| 场景 | 图片并发 | 视频并发 |
|------|:--------:|:--------:|
| 免费额度 / 低速 API | 1–2 | 1 |
| 标准付费账号 | 3–5 | 2–3 |
| 企业级账号 | 5–8 | 3–5 |

> 视频生成任务较重，并发过高容易触发服务商限流（429 错误）。建议从低并发开始，稳定后逐步调高。
