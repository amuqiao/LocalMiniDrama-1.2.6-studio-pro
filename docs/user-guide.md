# 使用指南：从零制作一部 AI 漫剧视频

> **文档职责**：手把手带领用户完成漫剧视频制作全流程，包含概念解释、按钮操作顺序和示例文本
> **适用场景**：首次使用本平台时阅读
> **目标读者**：无技术背景的创作者，按步骤操作即可完成第一部漫剧
> **维护规范**：界面按钮文字或操作顺序变更时同步更新本文档对应章节

---

## 先搞清楚三个概念

在开始操作之前，先理解三个容易混淆的词：

### 分镜
把一集剧本拆分成多个"镜头片段"，每个片段叫一个**分镜**。

比如剧本里写"林凡在竹林里发现白狐，蹲下来帮它松开铁夹"，AI 会把它拆成：
- 分镜 1：林凡走入竹林，听到声音
- 分镜 2：镜头下移，白狐被铁夹夹住
- 分镜 3：林凡蹲下，小心撬开铁夹

**分镜只是文字描述**，包含：场景、对白、动作、景别（全景/近景等）、时长。

### 分镜图
根据每个分镜的文字描述，用 AI **生成一张静态图片**，作为该镜头的画面参考。

> 类比：就像漫画里每一格的画面。

### 分镜视频
根据分镜图，用 AI **生成一段短视频**（通常 4–8 秒），让静态画面动起来。

> 类比：把漫画每一格变成动画帧。

### 三者的关系

```
剧本文字
  ↓ AI 拆解
分镜（文字描述 × N 个）
  ↓ AI 出图
分镜图（静态图片 × N 张）
  ↓ AI 生视频
分镜视频（短视频 × N 段）
  ↓ FFmpeg 拼接
完整视频（一集成片）
```

## 开始之前

确认以下条件已满足：

- 服务已启动：运行 `./service.sh start`，访问 [http://localhost:3013](http://localhost:3013) 能看到首页
- AI 配置已完成：右上角点击「**AI 配置**」，至少配置了 **text（文本）** 服务

> 配置 AI 服务参见 [AI 模型配置指南](./ai-config-openai.md)。最低只需 text 服务就能生成剧本和分镜文字；配置 image + video 后才能出图出片。

本文用以下示例贯穿全程：

> **《竹林奇缘》** — 少年林凡救下受伤白狐，白狐化为少女报恩，妖界武士突然破门追杀，二人一起逃亡。

## 完整操作顺序

### 第一步：新建项目

1. 打开首页，点击 **「+ 新建项目」**
2. 填写弹窗：

   | 字段 | 填写 |
   |------|------|
   | 标题 | 竹林奇缘 |
   | 描述 | 古风奇幻，少年与妖狐的逃亡故事 |
   | 画面比例 | 9:16（竖屏，适合短视频） |

3. 点击「确定」，进入剧目管理页

---

### 第二步：配置剧集风格

进入剧目管理页，在上方「剧集信息」卡片中：

1. **画面风格** → 选择 `日本动漫`
2. **故事梗概** → 填写：
   ```
   少年林凡救下受伤白狐，白狐化为少女报恩，妖界派人追杀，二人踏上逃亡之旅。
   ```
3. 点击「**保存**」

---

### 第三步：新建一集

在分集列表区点击 **「+ 新增一集」**，点击新建的「第 1 集」卡片，进入**制作页**。

> 后续所有操作都在这个制作页完成，页面从上到下就是操作顺序。

---

### 第四步：生成剧本

**位置**：制作页顶部「故事生成」卡片

填写以下内容：

| 字段 | 填写 |
|------|------|
| 故事梗概 | 少年林凡在竹林砍柴时，发现一只被猎人夹子夹伤的白狐。他解救了白狐，当晚白狐化身少女姜瑶登门感谢。就在二人相谈时，追杀姜瑶的妖界武士突然破门而入，林凡和姜瑶不得不一起逃离。 |
| 故事风格 | 古风 |
| 剧本类型 | 剧情 |
| 生成集数 | 1 |

点击 **「生成剧本」**，等待 15–40 秒。

生成后，剧本会出现在下方「剧本」卡片的文本框里，可以直接修改。确认内容后点击 **「保存当前集」**。

---

### 第五步：生成角色图和场景图

**位置**：制作页中部「资源管理」卡片（点击标题可展开/收起）

这一步让 AI 从剧本里识别角色和场景，并生成对应的参考图。参考图会在后续生成分镜图和视频时自动使用，保持人物形象一致。

#### 角色生成

1. 展开「**角色生成**」区块
2. 点击 **「剧本自动提取角色」** → AI 自动识别出林凡、姜瑶等角色
3. 点击每个角色卡片上的 **「编辑」**，在「外貌」字段补充描述（描述越详细，生成图越准确）：

   **林凡外貌示例：**
   ```
   十七岁少年，身形修长，五官清朗，黑色长发用布条束起，
   身穿灰色粗布短打，眼神坚定温和。
   ```

   **姜瑶外貌示例：**
   ```
   十六七岁白衣少女，长发如瀑，肌肤莹白，双眸清澈灵动，
   腰间系银白腰带，气质出尘。
   ```

4. 编辑完后，点击角色卡片上的 **「AI 生成」** 为该角色出图

#### 场景生成

1. 展开「**场景生成**」区块
2. 点击 **「剧本自动提取场景」** → AI 识别出竹林、茅屋等场景
3. 点击每个场景的 **「编辑」**，在「图片提示词」字段填写细节：

   **竹林场景提示词示例：**
   ```
   古风竹林，清晨薄雾弥漫，阳光透过竹叶洒落斑驳光影，
   远处隐约可见山峦，宁静空灵，高清写实。
   ```

4. 点击场景卡片上的 **「AI 生成」** 为该场景出图

> 道具生成可选做，对视频影响较小，初次使用可跳过。

---

### 第六步：生成分镜

**位置**：制作页「分镜生成」卡片

点击 **「AI 生成分镜」**，等待 30–60 秒。

AI 会把剧本拆解成 7–10 个分镜，每个分镜自动关联对应的角色和场景（不需要手动选择）。示例效果：

| 分镜 | 描述 | 对白 | 景别 |
|------|------|------|------|
| #1 竹林晨雾 | 林凡背竹篓走入薄雾竹林 | 无 | 全景 |
| #2 发现白狐 | 听到哀鸣，拨开竹丛，白狐被铁夹夹住 | 无 | 中景→近景 |
| #3 救治白狐 | 林凡蹲下撬开铁夹，撕衣包扎 | 「别怕，我来帮你。」 | 近景 |
| #4 白狐消失 | 白狐抬眼凝视林凡，消失在竹影中 | 无 | 特写 |
| #5 夜叩茅屋 | 夜晚茅屋，有人叩门 | 无 | 全景 |
| #6 姜瑶现身 | 白衣少女立于月色中 | 「公子救命之恩，姜瑶来报。」 | 中景 |
| #7 武士破门 | 黑衣武士持剑破门而入 | 「妖狐姜瑶，就地格杀！」 | 中景 |
| #8 逃亡开始 | 姜瑶抓住林凡的手，二人冲出门去 | 「走！」 | 全景→推进 |

分镜生成后，可点击每个分镜右上角的 **「编辑」** 修改对白或调整镜头参数。

---

### 第七步：生成分镜图

**位置**：分镜列表上方

确认分镜文字内容无误后，点击 **「批量生成分镜图」**。

系统自动为所有分镜生成静态图片，进度实时显示。每张图约 10–30 秒，8 个分镜约需 3–8 分钟。

生成完成后，逐个检查图片是否符合预期。如有问题，点击该分镜图旁的 **「重新生成」** 单独重试。

---

### 第八步：生成分镜视频

**位置**：分镜列表上方

所有分镜图确认后，点击 **「批量生成分镜视频」**。

系统自动为每张分镜图生成对应的短视频（每段约 5 秒），并行处理，进度实时显示。每段视频约 1–5 分钟，8 个分镜共需约 10–40 分钟。

> 视频生成期间不用守候，可以关闭页面，回来刷新进度即可。

---

### 第九步：合成完整视频

**位置**：制作页底部「合成视频」卡片

所有分镜视频完成后，点击 **「合成视频」**。

系统将所有分镜视频按顺序拼接为一个完整 MP4，约需 10–30 秒。完成后页面显示视频预览，可直接下载。

## 两种使用方式

### 方式 A：全自动（新手推荐）

写好剧本后，直接点 **「一键成片带图片视频」**，剩下的步骤全部自动完成。

适合：快速出片，对每个细节要求不高。

### 方式 B：分步控制

按照第四步到第九步依次操作，每步可以检查和修改再进行下一步。

适合：对角色外貌、场景风格有特定要求，想精细控制每个镜头。

## 常见问题

**Q：生成的角色图里人物和后面分镜图里的人物长得不一样？**

在「角色生成」中把角色外貌描述写得越详细越好（包括发型、服装颜色、体型等）。推荐使用火山引擎的 `seedream-3-0` 作为分镜图模型，它的参考图一致性最好。

**Q：分镜视频生成失败？**

点击页面右上角「任务」查看错误详情。最常见的原因是 API 余额不足或服务商限流，等几分钟后点分镜上的「**重新生成**」即可。

**Q：想要画面比例是横屏（16:9）？**

在第一步新建项目时，画面比例选 `16:9`。已创建的项目可以在「剧集信息」里修改，但已生成的图片和视频不会自动更换比例，需要重新生成。

**Q：剧本生成结果不满意？**

修改「故事梗概」，加入更多细节（主要角色名字、核心冲突、结局），然后重新点「生成剧本」。通常 2–3 次能得到满意的版本，也可以直接手动修改文本框里的内容。
